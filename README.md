## Description
Vitrin is a sample app shows subcategories and items of certain categories. 



## Key Features

* Smooth Scroll to Top
* Add Item to Cart With Animation
* Custom Drawer as Cart  
* Sectioned RecyclerView
* Animated Item Detail
